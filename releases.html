Installing & Updating TOHO

Steam Installation

Step 1: Download the ZIP file from the latest release that reads TOHO-Steam, and extract the files into a new folder.

Step 2: Right click on Among Us in the steam library. Select Browse Local Files from the pop-up.

Step 3: Copy and paste all files from the folder that pops up into your extracted TOHO folder. Run among us by double clicking on the Among Us file.

Epic Installation

Step 1: Download the ZIP file from the latest release that reads TOHO-Epic, and extract the files into a new folder.

Step 2: Copy and paste all files from your extracted TOHO folder and into the C:/Program Files/Epic Games/AmongUs folder.

Step 3: Run Among Us from the Epic Games launcher.

Updating TOHO

Step 1: Download the DLL file from the latest release that reads TOHO.dll.

Step 2: Visit your previously installed TOHO folder, and open the BepInEx/plugins folder. Replace the existing TOHO.dll with your new one.

Step 3: Run Among Us.
